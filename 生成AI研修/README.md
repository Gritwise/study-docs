# 生成AI研修スライドプロジェクト

このプロジェクトは [Slidev](https://sli.dev/) を使用して作成された、コンサルタント向けの生成AI研修用スライドです。

## セットアップ

このプロジェクトを実行するには [Node.js](https://nodejs.org/) がインストールされている必要があります。

1. リポジトリをクローンまたはダウンロードします。
2. 依存関係をインストールします（初回のみ）。

```bash
npm install
```

## 起動方法

以下のコマンドを実行して、ローカルの開発サーバを起動します。

```bash
npx @slidev/cli
```

起動後、ブラウザで `http://localhost:3030/` にアクセスすることでスライドを閲覧できます。

## プロジェクト構成

- `slides.md`: スライドのメインコンテンツ（Markdown形式）
- `bg-gritwise.png`: 背景画像
- `extract_pdf.js`: 資料（PDF）からテキストを抽出するために使用したユーティリティスクリプト
- `package.json`: プロジェクトの依存関係設定

## スライドの編集

`slides.md` を編集すると、ブラウザ上の表示がリアルタイムで更新されます（ホットリロード）。

## PDF書き出し

スライドをPDFとして書き出す場合は、以下のコマンドを使用します。

```bash
npx @slidev/cli export
```

※ 初回実行時や、PDF生成に必要なブラウザが見つからない場合は、以下のコマンドで依存関係をインストールしてください。

```bash
npm install -D playwright-chromium
```
